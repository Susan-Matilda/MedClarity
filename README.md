# MedClarity 🏥

## Overview

MedClarity is a healthcare web application that helps patients understand complex medical reports in simple language. The platform provides an interactive and user-friendly interface for uploading reports, viewing health summaries, listening to explanations, and locating nearby healthcare facilities.

Developed as part of the Full Stack Development (MCA413-1) CIA project at CHRIST (Deemed to be University).

---

## Problem Statement

Many patients receive medical reports containing technical medical terminology that is difficult to understand. This can lead to confusion, anxiety, and poor health decisions.

MedClarity aims to bridge this gap by simplifying medical information and presenting it in an accessible and interactive manner.

## Features

### Core Features

* Upload medical reports
* Simplified explanation of medical terms
* Health dashboard visualization
* Nearby hospital locator
* Save report history and preferences

### Accessibility Features

* Text-to-Speech for report explanations
* Easy-to-read healthcare summaries

### Interactive Features

* Drag & Drop report upload
* Dynamic health dashboard
* Form validation and event handling

## HTML5 APIs Used

### File API

* Upload and read medical reports

### Geolocation API

* Detect user location
* Find nearby hospitals

### Local Storage API

* Save user preferences
* Store report history

### Web Speech API

* Read report explanations aloud

### Canvas API

* Visualize health metrics and health score

### Bonus: Drag & Drop API

* Drag files directly into the upload area
